# Task Manager Palacios

Aplicación web para gestionar tareas, permitiendo registrar, organizar y dar seguimiento a actividades pendientes del usuario.

<!-- BADGE_CI -->

## Instalación local 🚀

```bash
git clone git@github.com:Jolcen/task_manager_palacios.git
cd task_manager_palacios
npm install
```

### Variables de entorno

Crea un archivo `.env` en la raíz del proyecto con las siguientes claves. No subas valores reales al repositorio.

```env
DATABASE_URL=
JWT_SECRET=
PORT=
```

## Comandos disponibles 📜

| Comando | Descripción |
|--------|-------------|
| `npm run dev` | Levanta el entorno de desarrollo |
| `npm run build` | Genera el build de producción |
| `npm test` | Corre las pruebas automatizadas, pendiente para la Sesión 3 |

## Base de datos 🗄️

PostgreSQL con migraciones y seeds gestionados con Prisma.
